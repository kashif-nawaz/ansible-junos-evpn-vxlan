This role will be executed across all the hosts listed in $PWD/inventory/{{site}}/hosts.yml

This role will execute junos operational commands and will be save the output as a text file in the ansible host 

Input file for this role is:-

- $PWD/inventory/{{site}}/group_vars/all/cli.yml

Output files generated by role are:-

- $PWD/host_vars/{{site}}/{{inventory_hostname}}/cli/{{junos-command}}.txt
