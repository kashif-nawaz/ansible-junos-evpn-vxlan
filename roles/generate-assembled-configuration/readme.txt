This role will assemble config files generated by all previous role and generate a single config file

Input files for this role are: -

- $PWD/host_vars/{{site}}/{{inventory_hostname}}/<file name> <date  & time stamp>. conf.part

Output file generated by this role is: -

- $PWD/host_vars/{{site}}/{{inventory_hostname}}/main. {{inventory_hostname}}/-config-<date & time stamp>.conf
- Config files generated during previous run will be moved to $PWD/host_vars/{{site}}/{{inventory_hostname}}/backup directory
